# Personal Portfolio & Resume Website

A responsive personal portfolio website that showcases my projects, technical skills, and professional experience.

The website is designed as a lightweight static site focused on performance, clean UI, and easy navigation.

## Live Demo

https://oleksandrteslenko.github.io/resume

## Repository

https://github.com/oleksandrteslenko/resume

## Features

* Responsive layout
* Project portfolio showcase
* Technology stack overview
* Links to live demos and GitHub repositories
* Optimized static website performance
* Clean and modern UI
* GitHub Pages deployment

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Installation

Clone the repository:

```bash
git clone https://github.com/oleksandrteslenko/resume.git
```

Open the project folder and launch `index.html` in your browser.

## Deployment

The website is deployed using GitHub Pages.
